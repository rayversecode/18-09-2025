Tables and Lists in HTML

First Principle: Structuring Content

Web content often needs to be structured into itemized lists or organized into a grid of rows and columns. HTML provides two primary tools for this: Lists and Tables.

Nested Lists

One of the most powerful features of HTML lists is the ability to "nest" them inside one another. This allows for the creation of complex outlines and sub-categories. You can place an unordered list (<ul>) inside an ordered list (<ol>), and vice-versa, to create detailed hierarchies.

Example: Nested Countries & Cities List

This example shows a top-level unordered list for countries. Inside each country's list item, there is another nested list for cities. Notice how "texas" has a further nested ordered list for its cities.

Example: Ordered Meal List

This example uses an ordered list as the main structure for meals (Breakfast, Lunch, Dinner). Inside "Breakfast," an unordered list is used to separate "main" items and "sides," showing how different list types can be combined for clarity.

Introduction to Basic Tables

Tables are used for displaying data in rows and columns. The basic tags are <table>, <tr> (table row), and <td> (table data/cell). The border attribute can be added to make the cell outlines visible.

Example: Basic Name/Role Table

Semantic Table Structure

For more structured and accessible tables, HTML provides <thead> (table head), <tbody> (table body), and <tfoot> (table foot). Header cells are defined with <th>, which makes them bold and centered.

Example: Structured Table (Note: the tag is correctly spelled <tfoot>)

Spanning Columns with colspan

The colspan attribute allows a single cell to stretch across multiple columns. In this example, the "Contact Info" header spans over two columns, grouping the phone and email data below it.

Example: Contact Info Table

Spanning Rows with rowspan and Combining Spans

The rowspan attribute allows a cell to stretch down over multiple rows. You can combine rowspan and colspan to create complex table layouts for displaying data.
